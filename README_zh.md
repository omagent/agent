---

<a id="核心特性"></a>

## ✨ 核心特性

- **🌐 内置浏览器**：在 OMAGT 内直接浏览网页、搜索信息、与网页交互，无需切换外部浏览器。
- **🧠 自动学习技能**：OMAGT 从你的工作流程中学习，自动创建新技能来处理重复性任务，持续扩展自身能力。
- **🚀 一键安装，开箱即用**：提供 Windows (.exe)、macOS (.dmg) 和 Linux (.AppImage) 预构建安装包，同时支持 Homebrew 安装。无需终端或编程知识。
- **🤖 灵活多模型支持**：支持 Claude、GPT、Gemini、DeepSeek、智谱 GLM、MiniMax、Kimi 等，兼容所有 OpenAI 格式 API。
- **🧰 内置 Skills 技能系统**：一键生成 PPTX、DOCX、XLSX、PDF 文档。支持自定义技能开发，内置 skill-creator 工具包。
- **🔌 MCP 外部工具集成**：通过 MCP 协议连接浏览器、Notion 等桌面应用，将 AI 能力扩展到文件管理和编程之外。

## ❓ 常见问题

**OMAGT 是什么？**
OMAGT 是一款开源的 AI 智能助手桌面应用，将 AI 模型（Claude、GPT、Gemini、DeepSeek 等）封装为图形界面，提供 Windows、macOS 和 Linux 一键安装包，无需命令行或编程知识。

**支持哪些 AI 模型？**
支持 Claude（通过 Anthropic 或 OpenRouter）、OpenAI 兼容接口，以及国产大模型包括智谱 GLM、MiniMax、Kimi 等。任何提供 OpenAI 兼容 API 的服务商都可以配置使用。

**数据安全吗？**
OMAGT 完全在本地运行，你的文件保留在你的工作区内。唯一的外部通信是与你配置的 AI 模型 API 之间的交互。没有任何数据被发送到 OMAGT 的服务器。

**支持 Linux 吗？**
支持。OMAGT 提供 Linux (x64) 预构建的 AppImage 安装包。
